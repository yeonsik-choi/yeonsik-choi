## 🚀 Projects

### 1. 사용자 위치 기반 운동 추천 챗봇 앱 개발
* **내용:** 사용자 위치(GPS) 기반으로 인근 운동 시설을 추천하고 정보를 제공하는 대화형 챗봇 어플리케이션 개발
* **성과:** 공공 데이터와 LLM을 결합하여 환각 현상을 억제한 신뢰도 높은 추천 시스템 구축

### 2. 체형 기반 옷장 활용 코디 추천 앱 개발
* **내용:** 사용자의 신체 사진을 분석하여 체형을 분류하고, 사용자가 보유한 옷장 속 의류를 조합하여 맞춤형 코디를 추천하는 모바일 애플리케이션 개발
* **성과:** 이미지 처리 기술(CV)을 활용하여 체형 분석부터 의류 분류까지 전 과정을 자동화한 서비스 구현

### 3. 온라인 거래 이상 탐지
* **내용:** 대규모 거래 데이터를 활용하여 온라인 결제 사기(Fraud)를 탐지하는 머신러닝 모델 개발
* **성과:** ROC-AUC 0.952 (Public) / 0.927 (Private) 달성, Stacking 앙상블을 통한 과적합 문제 해결

### 4. 아동, 청소년 게임 이용 부정적 인식 개선 분석
* **내용:** 패널 데이터를 활용하여 게임 행동 유형이 아동·청소년의 삶에 미치는 영향을 실증적으로 분석
* **성과:** 게임 이용이 대인관계 형성에 긍정적 영향을 미침을 통계적으로 입증

### 5. 동국대학교 파이썬 프로그래밍 챗봇
* **내용:** 강의안 데이터를 기반으로 한 질의응답 챗봇 구현 및 Gradio UI 시각화
* **성과:** 유사도 분석 알고리즘을 도입하여 오타나 문장 형태의 변화에도 유연하게 대처하도록 설계
